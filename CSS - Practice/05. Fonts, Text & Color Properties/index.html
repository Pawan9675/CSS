Understanding Fonts in CSS

This guide explains everything about fonts and text properties in CSS.

External Fonts Usage

We can use external fonts in two ways:

  • Using <link> in the <head>
  • Using @import inside the CSS file

Font Properties

1. Font Family

Defines the typeface to be used. Example: font-family: 'Poppins', sans-serif;

2. Font Size

Sets the size of the text. Example: font-size: 20px;

3. Font Weight

Controls the thickness of text. Example: font-weight: bold;

4. Font Style

Defines whether the text is normal, italic, or oblique. Example: font-style: italic;

5. Line Height

Adjusts the space between lines. Example: line-height: 1.5;

6. Letter and Word Spacing

Adjusts the spacing between letters and words.

  • Letter spacing: letter-spacing: 1px;
  • Word spacing: word-spacing: 3px;
