CSS Shadows and Outlines

Box Shadows

The box-shadow property allows designers to add shadow effects around an element's frame. It is used to give a 3D feel or to emphasize elements on hover.

Syntax:

box-shadow: h-offset v-offset blur spread color inset;
  • h-offset: Determines the shadow's horizontal position.
  • v-offset: Determines the shadow's vertical position.
  • blur: The larger the value, the blurrier the shadow.
  • spread: Expands or shrinks the shadow size.
  • color: Defines the shadow color.
  • inset: Makes the shadow inner.

Example:

.box-shadow-example { box-shadow: 5px 5px 15px 5px #888888; }

Text Shadows

The text-shadow property is used to give shadows to text, enhancing readability or creating a unique appearance.

Syntax:

text-shadow: h-offset v-offset blur color;

Example:

.text-shadow-example { text-shadow: 2px 2px 4px #888888; }

This is text with a shadow.

Outlines

The outline property defines a line drawn around elements, outside the borders, to make the element stand out. It's often used for accessibility purposes, like highlighting focused elements.

Syntax:

outline: width style color;
  • width: Sets the outline width.
  • style: Determines the style of the outline (solid, dotted, dashed, etc.).
  • color: Defines the outline color.

Example:

.outline-example { outline: 2px solid blue; }

Differences between Outlines and Borders

Although outlines and borders can look similar, there are key differences:

  • Position: Outlines don't take up space; they are drawn outside the element.
  • Offset: Using outline-offset, you can control the space between an outline and the element.
  • Width: Borders can have different widths on each side, while outlines are uniform.
  • Rounded Corners: Borders support border-radius, while outlines generally do not.

Example: Border vs Outline

Border Example:

.border-example { border: 4px solid blue; }
