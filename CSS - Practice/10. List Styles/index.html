CSS List Styles

Unordered List Styling

The list-style-type property is used to define the type of bullet point for unordered lists.

Syntax:

list-style-type: value;
  • disc: Default filled circle.
  • circle: Hollow circle.
  • square: Filled square.
  • none: Removes the bullet points.

Example:

.ul-disc { list-style-type: disc; }
  • Disc marker (default)
  • This is an unordered list
  • Circle marker
  • This is another unordered list
  • Square marker
  • Example with square bullets
  • No marker
  • Bullets removed

Ordered List Styling

The list-style-type property also defines the numbering system for ordered lists.

Syntax:

list-style-type: value;
  • decimal: Default 1, 2, 3, ...
  • decimal-leading-zero: 01, 02, 03, ...
  • lower-roman: i, ii, iii, ...
  • upper-roman: I, II, III, ...
  • lower-alpha: a, b, c, ...
  • upper-alpha: A, B, C, ...

Example:

.ol-decimal { list-style-type: decimal; }
  1. Decimal format
  2. Default numbering
  1. Leading zeros
  2. Formatted as 01, 02, ...
  1. Lowercase Roman numerals
  2. Example: i, ii, iii...
  1. Uppercase Roman numerals
  2. Example: I, II, III...
  1. Lowercase alphabet
  2. Example: a, b, c...
  1. Uppercase alphabet
  2. Example: A, B, C...

List-Style Position

The list-style-position property controls whether the bullet/number appears inside or outside the content box.

Syntax:

list-style-position: inside | outside;
  • inside: The marker is inside the text content.
  • outside: The marker is placed outside the content box (default).

Example:

.ul-inside { list-style-position: inside; }
  • Inside position
  • Marker inside content box
  • Outside position
  • Marker outside content box

Removing Default List Styles

The list-style property can be set to none to remove default bullets or numbering.

Syntax:

list-style: none;

Example:

.ul-remove { list-style: none; }
  • No list marker
  • Bullets are removed
