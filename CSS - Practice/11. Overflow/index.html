CSS Overflow Property 🎨

📌 What is Overflow?

Sometimes, content inside an element is larger than its container. The overflow property decides how to handle such situations. It controls whether the extra content should be **visible, hidden, scrollable, or auto-managed**.

🔹 Overflow: Visible (Default Behavior)

Content that exceeds the box remains fully visible.

This text overflows out of the box! 🚀

🔹 Overflow: Hidden

Excess content is **cut off** and hidden.

This text is too long but will be hidden...

🔹 Overflow: Scroll

Always shows a scrollbar, even if content fits.

This text is scrollable. Try scrolling! 🎡

🔹 Overflow: Auto

Only shows a scrollbar if content overflows.

Scrollbars appear **only when needed**.

🔹 Overflow: Shorthand (X & Y)

Setting `overflow` for X and Y separately.

X has scroll if needed, Y is hidden! 🌍
