ðŸ”¹ What are Media Queries?

Media queries allow you to apply different styles based on screen size, orientation, and device type. They make websites responsive by adjusting layouts dynamically.

ðŸ”¹ Syntax of a Media Query

        @media media-type and (condition) {
            /* CSS rules */
        }

ðŸ”¹ Key Features of Media Queries

  • max-width: Applies styles when the screen width is smaller than a given value.
  • min-width: Applies styles when the screen width is larger than a given value.
  • orientation: Targets specific orientations (portrait/landscape).
  • only: Ensures styles apply only to certain media types.
  • not: Excludes specific conditions from applying styles.
  • print: Applies styles when printing the page.

ðŸ”¹ Different Media Types

  • screen: Applies styles to screens (monitors, smartphones, tablets).
  • print: Used for printed documents.
  • all: Applies styles to all devices.
  • speech: Targets screen readers.

ðŸ”¹ Combining Conditions

  • Using and: @media screen and (max-width: 600px) and (orientation: portrait)
  • Using or (comma separated): @media screen and (max-width: 600px), (orientation: portrait)
  • Using not: @media not screen and (max-width: 600px)
