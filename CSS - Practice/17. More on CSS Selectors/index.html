The foundation of CSS selection - the most commonly used selectors.

Universal Selector (*)

Selects all elements. In this page, it sets margin: 0; padding: 0; box-sizing: border-box; for everything.

* { margin: 0; padding: 0; box-sizing: border-box; }

Element Selector

This text is styled using the element selector - all paragraphs are set to darkblue with 18px font-size.

p { color: darkblue; font-size: 18px; }

ID Selector (#)

This paragraph has the ID "special" - notice the red color and bold text.

#special { color: red; font-weight: bold; }

Class Selector (.)

This paragraph has the "highlight" class - with yellow background.

.highlight { background-color: yellow; padding: 5px; }

Grouping Selector

Notice how h1, h2, and p elements all use Arial font - this is from the grouping selector.

h1, h2, p { font-family: Arial, sans-serif; }

2Ô∏è‚É£ Combinator Selectors

These selectors define relationships between elements.

Descendant Selector (space)

This is a paragraph inside .container - notice it's green because of the descendant selector.

This paragraph is inside a div, which is inside .container - still green!

.container p { color: green; }

Child Selector (>)

This is a direct child paragraph - it's blue due to the child selector.

This paragraph is NOT a direct child - so it stays green from the descendant selector.

.container > p { color: blue; }

Adjacent Sibling Selector (+)

This is an h2 element

This paragraph is immediately after an h2 - it's brown due to adjacent sibling selector.

This paragraph comes later - not immediately after h2.

h2 + p { color: brown; }

General Sibling Selector (~)

This is another h2 element

This paragraph is after h2 - it's italic due to general sibling selector.

This paragraph is also after h2 - it's also italic!

h2 ~ p { font-style: italic; }

These selectors target specific parts of elements.

5Ô∏è‚É£ Attribute Selectors

These selectors target elements based on their attributes.

[attribute]

This div has a data-color attribute - it's darkred.
[data-color] { color: darkred; }

[attribute=value]

input[type="text"] { border: 1px solid blue; }

[attribute~=value]

This div has "important" in its title attribute.
[title~="important"] { font-weight: bold; color: red; }

[attribute|=value]

This div has lang="en-US" - it matches [lang|="en"].
[lang|="en"] { font-style: italic; }

[attribute^=value]

This link starts with "https" - it's green.
a[href^="https"] { color: green; }

[attribute$=value]

This link ends with ".pdf" - it's red and bold.
a[href$=".pdf"] { color: red; font-weight: bold; }

[attribute*=value]

This link contains "docs" - it's underlined.
a[href*="docs"] { text-decoration: underline; }
