This guide demonstrates all the key aspects of CSS Flexbox layout with practical examples.

1. Basic Flexbox

Flexbox is a one-dimensional layout method designed for laying out items in rows or columns. To create a flex container, set display: flex on an element. Its direct children automatically become flex items.

9. Flex Item Properties

order

The order property controls the order in which flex items appear in the flex container. Default is 0.

1 (default: 0)
2 (order: 1)
3 (order: 2)
4 (order: -1)

flex-grow

The flex-grow property defines the ability for a flex item to grow if necessary. Default is 0. It specifies what amount of the available space inside the flex container the item should take up.

Item 1 (flex-grow: 0)
Item 2 (flex-grow: 1)
Item 3 (flex-grow: 2)

flex-shrink

The flex-shrink property defines the ability for a flex item to shrink if necessary. Default is 1.

Item 1 (flex-shrink: 0)
Item 2 (flex-shrink: 1)
Item 3 (flex-shrink: 2)

flex-basis

The flex-basis property defines the default size of an element before the remaining space is distributed. Default is auto.

Item 1 (flex-basis: auto)
Item 2 (flex-basis: 0)
Item 3 (flex-basis: 100px)
Item 4 (flex-basis: 50%)

flex (shorthand)

The flex property is a shorthand for flex-grow, flex-shrink, and flex-basis combined.

Item 1 (flex: 0 1 auto - default)
Item 2 (flex: 1 1 auto)
Item 3 (flex: 1)
Item 4 (flex: auto)
Item 5 (flex: none)

align-self

The align-self property allows the default alignment (set by align-items) to be overridden for individual flex items.

Summary of Key Flexbox Properties

Flex Container Properties:

  • display: flex - Defines a flex container
  • flex-direction - Direction of flex items (row, row-reverse, column, column-reverse)
  • flex-wrap - Whether items wrap (nowrap, wrap, wrap-reverse)
  • flex-flow - Shorthand for flex-direction and flex-wrap
  • justify-content - Alignment along main axis
  • align-items - Alignment along cross axis
  • align-content - Alignment of wrapped lines
  • gap, row-gap, column-gap - Spacing between items

Flex Item Properties:

  • order - Controls order of items
  • flex-grow - How much item can grow
  • flex-shrink - How much item can shrink
  • flex-basis - Default size before distributing space
  • flex - Shorthand for grow, shrink and basis
  • align-self - Override align-items for specific item
