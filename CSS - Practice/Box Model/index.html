Key Concepts of the CSS Box Model

The CSS Box Model defines how elements are structured and how dimensions are calculated. It consists of:

  • Content: The innermost part of the box, containing text, images, or other elements.
  • Padding: Space between the content and the border.
  • Border: The boundary surrounding the padding and content.
  • Margin: The outermost space separating the element from others.

Calculating Total Dimensions

The total dimensions of an element are calculated as:

Total Width = Content Width + Left Padding + Right Padding + Left Border + Right Border + Left Margin + Right Margin

Total Height = Content Height + Top Padding + Bottom Padding + Top Border + Bottom Border + Top Margin + Bottom Margin

Example:


p {
    width: 200px;
    height: 300px;
    padding: 15px;
    border: 10px solid red;
    margin: 5px;
}

Calculated dimensions:

  • Total Width: 200px (content) + 15px (left padding) + 15px (right padding) + 10px (left border) + 10px (right border) + 5px (left margin) + 5px (right margin) = 260px
  • Total Height: 300px (content) + 15px (top padding) + 15px (bottom padding) + 10px (top border) + 10px (bottom border) + 5px (top margin) + 5px (bottom margin) = 360px

Margin Collapse

When two vertical margins meet, they collapse into a single margin. The size of the collapsed margin is equal to the largest of the two margins.
