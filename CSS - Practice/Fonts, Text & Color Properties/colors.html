Let's Learn About Colors

Color can be represented in the following ways in CSS:

  1. Color Keywords
  2. Hex Color Code
  3. RGB
  4. RGBA
  5. HSL

Color Representation Methods

1. Color Keywords

Color keywords are the simplest way to specify colors. These predefined names represent a wide variety of colors. Examples of color keywords include:

  • red
  • blue
  • green
  • yellow
  • black
  • white

2. Hexadecimal Color Code

The hexadecimal color code uses six digits to define the color. The first two digits represent the red value, the second two represent green, and the last two represent blue.
For example: #FF5733 represents an orange color, where:

  • FF = Red component (255 in decimal)
  • 57 = Green component (87 in decimal)
  • 33 = Blue component (51 in decimal)

3. RGB (Red, Green, Blue)

In RGB, colors are represented as rgb(red, green, blue) where each value is between 0 and 255. Example: rgb(76, 125, 59) creates a greenish color.
Here, 76 is the red value, 125 is the green value, and 59 is the blue value.

4. RGBA (Red, Green, Blue, Alpha)

RGBA is similar to RGB, but it also includes an alpha value for transparency. The alpha value ranges from 0 (fully transparent) to 1 (fully opaque). Example: rgba(255, 22, 5, 0.175) represents a red color with 17.5% opacity.

5. HSL (Hue, Saturation, Lightness)

In HSL, colors are represented using three values:

  • Hue: The color type, measured in degrees from 0 to 360 (e.g., 0 is red, 120 is green, 240 is blue)
  • Saturation: The intensity of the color, represented as a percentage (0% is gray, 100% is full color)
  • Lightness: The brightness of the color, also represented as a percentage (0% is black, 100% is white)
Example: hsl(130, 61%, 44%) is a green color with:
  • Hue: 130 (green)
  • Saturation: 61%
  • Lightness: 44%
